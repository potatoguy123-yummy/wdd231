Planning

Site Name

The site name is "EW private server". This is shown as "EW webui" in some places. This website name was chosen was because the backend server is named "ew" and this is the webui that will be embedded into the server.

Site Purpose

The purpose of this website is to give users an ability to browse cards/songs available in-game, and allows users to manage their account. Most of this website is primarily intended towards people that already have an account and know what they're doing. I possibly have plans in the future to write an install guide (move it out of the FAQ) but this is for later after this assignment if I ever do it.

I will be using Alegreya. I will use a larger weight for the header and footer, and a smaller weight for the body. I chose this font because it looks neat and a little playful and the feel matches with the color scheme and content.
